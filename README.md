# Cpu-Monitor-GUI-App
A modern Python desktop app that displays detailed CPU information — including per-core usage, frequency, cache size, and multi-thread benchmark rating — using Tkinter, psutil, and py-cpuinfo.

The application displays the usage rates per core, processor frequency, L3 cache size, and your CPU's multi-thread score using the Tkinter, psutil, and py-cpuinfo libraries.
